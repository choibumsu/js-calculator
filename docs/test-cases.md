# Test Cases

## 1. 2개의 숫자에 대해 덧셈이 가능하다.

### 1-1

**Given**

- 유저에게 화면이 렌더링된 후

**When**

- 숫자 1을 클릭
- \+ 버튼을 클릭
- 숫자 2를 클릭
- = 버튼을 클릭

**Then**

- 계산기 화면에 숫자 3이 표시된다.

### 1-2

**Given**

- 유저가 다른 연산을 진행한 직후
- 계산기 화면에는 숫자 3이 표시되어 있는 상태

**When**

- \+ 버튼을 클릭
- 숫자 3를 클릭
- = 버튼을 클릭

**Then**

- 계산기 화면에 숫자 6이 표시된다.
